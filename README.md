# Automated Report Generation System

An end-to-end system for automatically generating reports, featuring modular design for file handling, user authentication, and ChatGPT API integration.

## Project Modules

- **chain/** – Handles the **backend logic for file upload and download**, managing report data and workflow on the server side.  
- **login/** – Implements **user login and registration**, and includes the **frontend web pages** for user interaction.  
- **flaskProject/** – Integrates with the **ChatGPT API**, providing AI-assisted features for generating or summarizing report content.

## Tech Stack

- **Frontend:** Vue.js, HTML, CSS, JavaScript, Element UI 
- **Backend:** Spring Boot, Java, Flask (for ChatGPT API integration), Python
- **Authentication:** Custom login and registration system  
- **AI Integration:** OpenAI ChatGPT API  

## Environment

- **Python Version:** 3.8+  
- **Java Version:** 17+ (for Spring Boot)  
- **Node.js Version:** 16+ (for Vue.js)  
- **OS:** Cross-platform (Windows / macOS / Linux)  
- **Dependencies:** Flask, Spring Boot libraries, Vue.js packages (see respective `requirements.txt` / `pom.xml` / `package.json`)
